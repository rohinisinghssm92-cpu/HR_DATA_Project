# HR_DATA_Project

This project is based on HR data analysis using Python.

## Objective
To analyze employee data and extract useful insights such as attrition, performance, and trends.

## Tools & Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Dataset
HR employee dataset in CSV format.

## Conclusion
This project helps in understanding HR analytics and decision making using data.
